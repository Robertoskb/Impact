🔧 Teste do Modo Debug - Simulado ENEM

📋 Instruções de Uso

O modo debug permite gerar respostas aleatórias automaticamente, facilitando o teste rápido do simulado.

🚀 Links para Teste

🎯 Abrir Simulado com Debug 📝 Abrir Simulado Normal
⚠️ Atenção: O modo debug é apenas para desenvolvimento e testes. As respostas geradas são aleatórias e não refletem conhecimento real.

🔧 Funcionalidades do Debug Panel

🔧 Painel Movível

O painel pode ser arrastado clicando e segurando a barra do título "🔧 Debug Panel". Pode ser posicionado em qualquer lugar da tela conforme necessário. Use o botão "−" / "+" para minimizar/expandir o painel.

🎲 Gerar Respostas Aleatórias

Preenche automaticamente todas as questões com respostas aleatórias (A, B, C, D ou E). Útil para testar rapidamente o cálculo de notas TRI.

🗑️ Limpar Respostas

Remove todas as respostas marcadas, limpando tanto visualmente quanto nos dados internos.

✅ Preencher Corretas (Teste)

Tenta preencher com as respostas corretas baseadas no meta.json. Só funciona se o gabarito estiver disponível para o ano/área selecionados.

💻 Ativação Manual via Console

Alternativamente, você pode ativar o debug via console do navegador (F12):

localStorage.setItem('debugMode', 'true'); location.reload();

📊 Como Testar

  1. Clique no link "Abrir Simulado com Debug" acima
  2. Configure um simulado (selecione ano, tipo, cor)
  3. Inicie o simulado
  4. Você verá o painel de debug no canto superior direito
  5. Use "Gerar Respostas Aleatórias" para preencher tudo automaticamente
  6. Finalize o simulado para ver as notas TRI calculadas

🎯 Cenários de Teste Sugeridos

  • Teste Rápido: Use respostas aleatórias para verificar se o cálculo TRI funciona
  • Teste de Performance: Use respostas corretas para verificar notas máximas
  • Teste de Limpeza: Teste a funcionalidade de limpar respostas
  • Teste de Áreas: Teste diferentes áreas (LC0, LC1, CH, CN, MT)
💡 Dica: O painel de debug pode ser minimizado clicando no botão "−" e reaparecerá automaticamente quando você abrir um novo simulado.
